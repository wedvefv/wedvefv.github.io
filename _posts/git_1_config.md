# 基于公司是代理上网的，所以配置windows下的.gitconfig文件如下：


```cpp
[http]
	proxy = 代理ip:代理port
[user]
	email = xxxxx@gmail.com
	name =xxxxx 
[gui]  
encoding = utf-8 #代码库统一使用utf-8  
```
###由于本人电脑是日语版OS，所以GIT终端显示中文字体是乱码，所以索性设置全部
###使用utf-8编码。
