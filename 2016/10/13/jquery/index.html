jquery选择器

jQuery 选择器
jQuery 选择器允许您对 HTML 元素组或单个元素进行操作。
jQuery 选择器基于元素的 id、类、类型、属性、属性值等”查找”（或选择）HTML 元素。 它基于已经存在的 CSS 选择器，除此之外，它还有一些自定义的选择器。
jQuery 中所有选择器都以美元符号开头：$()。

元素选择器 ：依据元素名

id选择器 依据元素id

class选择器 依据class属性

语法 描述
$(“*”) 选取所有元素
$(this) 选取当前html元素
$(“p.intro”) 选取class是intro的p元素
$(“p:first”) 选取第一个p元素
$(“ul li:first”) 选取第一个ul元素的第一个li元素
$(“ul li:first-child”) 选取每个ul的第一个li
$(“[href]”) 选取带有href属性的元素
$(“a[target=’blank’]”) 所有target属性是blank的a元素
$(“a[target!=’blank’]”) 所有target属性不是blank的a元素
$(“:button”) 选取所有 type=”button” 的 元素 和
$(“tr:even”) 偶数行
$(“tr:odd”) 奇数行
