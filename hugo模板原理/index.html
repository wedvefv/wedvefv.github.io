hugo模板的基本语法

  • 标签 {{ }} ， 包含的内容叫做动作。
  • . 表示传递给模板的数据, 表示模板的上下文。
  • {{ /*comment*/ }} 注释
  • {{ - pipeline - }} 清除前后的空格
  • {{ - pipeline }} 清除前面的空格
  • {{ pipeline - }} 清除后面的空格
  • {{ $a := ‘10’ }} 变量赋值, go语法
  • {{ if pipeline }} T1 {{ end }} , 如果pipeline不为空，则输出T1。
  • {{ if pipeline }} T1 {{ else }} T0 {{end} 为空 则T1， 不为T0。
  • {{ if pipeline }} T1 {{ else if pipeline }} T0 {{ end }} .
  • {{ range pipeline }} T1 {{ end }}
